# 🌡️ Temperature Converter Application

A professional console-based Temperature Converter application built using JavaScript and Node.js.

This application allows users to convert temperatures between Celsius and Fahrenheit through an interactive CLI (Command Line Interface).

---

# 🚀 Features

- 🌡️ Celsius to Fahrenheit conversion
- 🔥 Fahrenheit to Celsius conversion
- 💻 Interactive CLI interface
- ⚠️ Input validation and error handling
- 🔁 Multiple conversion support
- 🧠 Clean and beginner-friendly code structure

---

# 🛠️ Technologies Used

- JavaScript
- Node.js
- Readline Module
